# 목표

교수 사용자는 자신의 강의를 운영하기 위한 학습 자료, 과제, 시험, 공지, 출석, 성적 기능을 사용할 수 있어야 한다.

# 필수 기능

- 강의자료 업로드
- 동영상 업로드
- 과제 생성 / 마감일 설정
- 시험 / 퀴즈 출제
- 학생 제출물 확인
- 채점 / 피드백
- 공지 작성
- 출석 / 진도율 확인
- 성적 관리

# 수용 기준

- 교수는 자신이 담당하는 강의 범위 안에서만 운영 기능을 수행할 수 있어야 한다.
- 과제와 시험은 시작 / 종료 시점과 대상 강의를 기준으로 제어되어야 한다.
- 출석과 진도율 조회는 강의 단위로 집계 가능해야 한다.

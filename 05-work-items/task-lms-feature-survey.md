# 조사 범위

- 학생 기능
- 교수 기능
- 관리자 기능
- 필수 / 선택 기능 분류
- Front 와 Backend 의 역할 분리 포인트

# 완료 기준

- 역할별 기능 목록 정리
- MVP 에 포함할 기능과 제외할 기능 구분
- 후속 구현 우선순위 제안

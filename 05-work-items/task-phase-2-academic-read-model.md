# 범위

- 로그인 페이지 분리
- 우측 상단 계정 요약과 프로필 페이지 분리
- 학생 강의 목록
- 교수 담당 강의 목록
- 공지사항 조회 / 상세 조회 / 작성
- 강의자료 / 동영상 임시 프론트 스캐폴드
- 관리자 사용자 목록
- 관리자 강의실 목록
- 관리자 강의실별 AP / Wi-Fi 매핑 목록
- 강의 상세 페이지와 우측 기능 바
- 학생 단말 관리는 프로필 페이지로 이동
- attendance / exam eligibility 는 강의 상세 페이지 내부에 배치

# 완료 기준

- 로그인 후 우측 상단에 간략한 계정 정보가 보인다.
- 프로필 페이지에서 학생 단말 등록 / 삭제 / 목록 확인이 가능하다.
- 학생 로그인 후 자신의 강의 목록을 볼 수 있다.
- 학생은 강의를 클릭해 강의 상세 페이지로 들어갈 수 있다.
- 강의 상세 페이지의 기본 메인은 공지사항이다.
- 학생은 공지 목록에서 상세 화면으로 이동할 수 있다.
- 교수 로그인 후 담당 강의 목록을 볼 수 있다.
- 교수는 공지사항을 작성할 수 있다.
- 교수는 자신이 조회 가능한 공지 상세를 확인할 수 있다.
- 강의자료 / 동영상 영역은 임시 스캐폴드로 구분되어 보이고, 세션 범위에서만 동작한다.
- 학생은 공지사항을 볼 수 있다.
- 관리자는 학생 / 교수 / 관리자 목록과 강의실 / AP 매핑 목록을 볼 수 있다.
